# NullsWorld-webgame-ui

NullsWorld-webgame-ui for PC.

### Run

Recommend to use pnpm, it's faster.

**0x0.Install pnpm**

```sh
npm install pnpm -g
```

**0x1.Install dependencies with pnpm**

```sh
pnpm i
```

**0x2.Run develop**

```sh
pnpm run dev
```

**0x3.Build release**

```sh
pnpm run build
```
