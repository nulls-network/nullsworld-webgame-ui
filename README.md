# Nulls-World

Nulls-World.

### Run

pnpm is recommended.

**Install pnpm**

```sh
npm install pnpm
```

**Install dependencies with pnpm**

```sh
pnpm i
```

**Run develop**

```sh
pnpm run dev
```

**Build release**

```sh
pnpm run build
```
